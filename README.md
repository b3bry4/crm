# CarService CRM (Backend API)

Backend API для CarService CRM на FastAPI.

## Кратко о проекте
- Основной сервис работает на VPS в Нидерландах.
- API обслуживается через `crm-api.service` и слушает только `127.0.0.1:8010`.
- Внешний доступ организован исключительно через отдельный Cloudflare Tunnel и DNS.
- Любые изменения должны быть изолированы в рамках CRM и не затрагивать другие продакшн‑сервисы.

## Инфраструктура (общее)
- Существующие продакшн‑сервисы на сервере менять или перезапускать нельзя.
- Основной Cloudflare‑туннель `paxatlink-nl` к CRM не относится и его трогать нельзя.
- Для CRM используется свой Cloudflare‑туннель и отдельный systemd‑сервис.

## Структура проекта
- `app.py` — точка входа FastAPI.

## Ограничения
- Не изменять `cloudflared.service`.
- Не менять и не перезапускать чужие сервисы.
- Работать только в рамках:
  - `crm-api.service`
  - `cloudflared-carservicecrm.service`
